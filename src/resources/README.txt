AVATAR IMAGES FOLDER
====================

This folder contains avatar images for team members displayed in the About panel.

HOW TO ADD AVATARS:
-------------------
1. Save team member photos in this folder
2. Name the files using the member's initials followed by .png, .jpg, or .jpeg

EXAMPLE FILE NAMES:
------------------
GHA.png  - George Harold A. Alcantara
AMA.jpg  - Aldrin Miguel A. Jariel
JCA.png  - John Christoper A. Perez
RPG.jpg  - Ron Paulo G. Angeles
MDD.png  - Matthew Dane D. Calangian

IMAGE REQUIREMENTS:
------------------
- Format: PNG, JPG, or JPEG
- Recommended size: 200x200 pixels or larger (square images work best)
- The image will be automatically cropped to a circle
- If no image is found, the system will display a colored circle with initials

FALLBACK BEHAVIOR:
-----------------
If an avatar image is not found, the system will automatically generate
a colored circle with the member's initials.
